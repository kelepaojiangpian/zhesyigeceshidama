# 锂电储能数智化教学演示系统 - 项目设计文档

## 1. 项目概述
本项目是一款专为教学演示设计的**锂电储能数字孪生监控大屏**。它模拟了一个智慧调度平台，通过高度直观、动态的视觉效果，展示储能系统在不同电力场景下的调度逻辑、AI 决策过程以及电池状态变化。

*   **项目名称**：锂电储能数智化教学演示系统 (Smart Energy Storage Dispatch System)
*   **核心定位**：教学演示、数字孪生模拟、AI 指令驱动展示

## 2. 设计理念
### 2.1 视觉风格 (Visual Identity)
*   **工业科技感 (Cyberpunk/Industrial)**：采用深色背景 (`#020617`) 配合霓虹色（青色、橙色、红色）作为点缀，营造出“指挥中心”或“赛博工厂”的氛围。
*   **几何装饰**：使用切角面板 (`industrial-panel`)、几何装饰角 (`corner-accent`) 以及背景扫描线 (`scanline`)，增强界面的专业度与精密感。

### 2.2 UI/UX 策略
*   **严格视口布局**：页面整体严格固定在浏览器视口内 (`100vw/100vh`)，禁用任何横向与纵向滚动条，确保在大屏展示或演示环境下信息完整均衡。
*   **响应式自适应**：通过 Tailwind CSS 的 Flexbox 布局以及 `min-h-0` 策略，使图表、卡片和面板能够根据视口大小自动伸缩，防止内容溢出。
*   **交互反馈**：利用高频震动、呼吸闪烁、流光光效等微动画，强化系统在“成功、警告、危险”三种状态下的视觉差异。

## 3. 技术栈
*   **框架**：Vue 3 (Composition API / <script setup>)
*   **构建工具**：Vite
*   **样式**：Tailwind CSS (响应式设计、JIT 模式)
*   **图表可视化**：ECharts (用于仪表盘监控与历史趋势分析)
*   **图标库**：Lucide-Vue-Next
*   **动画库**：原生的 CSS Keyframes 与 Tailwind Transitions

## 4. 核心功能与布局
### 4.1 监控看板 (Digital Twin)
*   **KPI 卡片组**：实时展示 SOC、剩余电量、运行功率和系统温度。
*   **仪表盘 (Gauge Chart)**：动态监控电池组核心电压波动。
*   **趋势图 (Trend Chart)**：关联展示功率负载与 SOC 的变化曲线。

### 4.2 智能调度中心 (AI Control Hub)
*   **场景切换**：内置三个教学演示场景：
    1.  **深夜谷电蓄能**：展示低价电量补给逻辑。
    2.  **中午高温观察**：演示负荷巅峰下的环境预警。
    3.  **负荷失衡预警**：模拟系统底层风险与 AI 紧急干预。
*   **AI 助手解释区**：实时输出当前场景的分析说明，辅助教学理解。
*   **数据流桥接 (JSON Bridge)**：展示驱动 UI 变化的底层 JSON 数据结构，方便技术原理讲解。

### 4.3 动态电池模型
*   **实时 SOC 映射**：可视化展示电池充放电深度。
*   **状态指示动画**：
    *   充电中：向右侧流光动画 (`flow`)。
    *   预警中：外壳呼吸光晕 (`shell-glow`)。
    *   危险中：高频物理震颤 (`battery-shake`)。

## 5. 项目约束
*   **禁止滚动**：`html, body` 强制 `overflow: hidden`。
*   **盒模型**：应用 `box-border` 确保 Padding 不撑破视口容器。
*   **性能**：针对长连接数据流模拟进行了防抖与定时刷新优化。
